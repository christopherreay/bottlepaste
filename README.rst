bottlepaste
-----------

A command line pastebin akin to ''http://sprunge.us/'' but using ''bottle.py''. The
authors deploys it on ''https://www.cloudcontrol.com/''.

Synposis
--------

Use curl to paste::

    $ cat bottlepaste.py | curl -F 'bp=<-'  $DEPLOYMENT_URL
    $ $DEPLOYMENT_URL/6e5da45
